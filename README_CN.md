# uni-react

`uni-react` 是一个面向三维分子预训练与性质预测的深度学习代码库。

这个仓库目前仍在持续开发中，现阶段主要包含模型实现、训练入口、辅助脚本和测试代码。

## 概览

```text
uni_react/
├── core/        接口定义
├── registry/    组件注册表
├── encoders/    backbone 与模型实现
├── heads/       任务头
├── losses/      损失函数
├── data/        数据处理与转换工具
├── trainers/    训练器实现
├── training/    通用训练工具
├── tools/       实用脚本
└── utils/       数据集与辅助模块
```

## 当前状态

- 研究中的实验性代码
- 接口和训练流程后续仍可能调整
- 文档目前保持简要说明
