# uni-react

`uni-react` 是一个面向三维分子预训练与性质预测的深度学习代码库。

这个仓库目前仍在持续开发中，现阶段主要包含模型实现、训练入口、配置与测试代码。

## 概览

```text
uni_react/
├── models/      模型单元（`single_mol`、`gotennet_l`）
├── tasks/       任务定义、数据集、任务头、loss、trainer 与入口
├── logger.py    训练日志与结果落盘
├── training/    通用运行时工具、调度器、训练基类与累加器
└── data/        数据处理与转换工具
```

## 当前状态

- 研究中的实验性代码
- 接口和训练流程后续仍可能调整
- 文档目前保持简要说明
