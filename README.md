# uni-react

`uni-react` is a deep learning codebase for 3D molecular pretraining and property prediction.

This repository is still under active development. The current codebase mainly includes model implementations, training entry points, helper scripts, and tests.

## Overview

```text
uni_react/
├── core/        Protocol-style interfaces
├── registry/    Component registry
├── encoders/    Backbone and model implementations
├── heads/       Task heads
├── losses/      Loss functions
├── data/        Data pipeline utilities and converters
├── trainers/    Trainer implementations
├── training/    Shared training utilities
├── tools/       Utility scripts
└── utils/       Dataset and helper modules
```

## Status

- Experimental research code
- Interfaces and training workflows may still change
- Documentation is intentionally brief for now
